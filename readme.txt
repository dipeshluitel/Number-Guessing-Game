Number Guessing Game
Overview
This is a simple command-line number guessing game where the player can choose between easy and hard difficulty levels. The goal is to guess a randomly generated number between 1 and 99 within a limited number of attempts.

Game Rules
Easy Level: You have 10 lives to guess the correct number.
Hard Level: You have 5 lives to guess the correct number.
Each time you make a guess, you will be informed whether the correct number is higher or lower than your guess. If you guess the correct number within the given lives, you win. If you run out of lives, you lose.

How to Play
Run the game script.
Choose the difficulty level by typing 'e' for easy or 'h' for hard.
Guess the number between 1 and 99.
Follow the prompts to continue guessing until you either guess the correct number or run out of lives.

Notes
Ensure you have Python installed on your system to run this script.
The script will clear the terminal screen after each guess to provide a cleaner user experience.
